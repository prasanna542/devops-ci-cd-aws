# DevOps CI/CD Automation Project

This project demonstrates a complete CI/CD pipeline using GitHub Actions, Terraform, Ansible, and AWS EC2.

## Objective
To automate application deployment from code commit to production using DevOps best practices.

## Tools & Technologies
- Git & GitHub
- GitHub Actions
- AWS EC2
- Terraform
- Ansible
- Linux & Shell Scripting

## High-Level Workflow
1. Developer pushes code to GitHub
2. CI pipeline validates code
3. Infrastructure is provisioned using Terraform
4. Application is deployed using Ansible
